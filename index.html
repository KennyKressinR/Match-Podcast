Backup — Exportar / Importar
Los datos se guardan automáticamente en este dispositivo/navegador. Para moverlos a otro celular o PC, o para tener una copia de seguridad, exportá un archivo de backup y luego importalo donde lo necesites.
Importar un backup reemplaza todos los datos actuales de este dispositivo por los del archivo. Te vamos a pedir confirmación antes de aplicarlo.
